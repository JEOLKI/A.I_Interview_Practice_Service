# A.I Interview Practice Service (인공지능 모의면접 서비스)

### introduce
AI_INTREVIEW는 다른 앱이나 프로그램을 설치할 필요 없이 언제 어디서든 간편하게 면접 트레이닝을 실시하고, 
면접 후 제공되는 분석리포트를 제공하여 나의 부족한점을 보완할 수 있게 도와주는  가능 인공지능 기반 모의 면접 지원 서비스입니다.
